Flex classes

Apply display utilities to create a flexbox container and transform direct children elements into flex items. Flex containers and items are able to be modified further with additional flex properties.

This is flex-box container. To enable display: flex, use class d-flex.

For small screen use class d-sm-flex. For large screen use class d-md-flex.

This is inline flex-box container. To enable display: flex, use class d-inline-flex.

For small screen use class d-sm-inline-flex. For large screen use class d-md-inline-flex.

Opacity

The opacity property sets the opacity level for an element.

The opacity level describes the transparency level, where 1 is not transparent at all, .5 is 50% visible, and 0 is completely transparent. Available options for small and large screens - .op-*-20, .op-*-50, .op-*-100 (* - sm or md)

20%
50%
100%
<div class="d-flex"> <div class="littlebox op-20">20%</div> <div class="littlebox op-50">50%</div> <div class="littlebox op-100">100%</div> </div>

Padding

Padding is used to create space around an element's content, inside of any defined borders.

.p-none
.pl-5
.p-10
.pt-5
.pt-10

.p-none - none padding at all

.pl-5 - 5px padding from left

.p-10 - 10px padding from all sides

.pt-5 - 5px padding from top

.pt-10 - 10px padding from top

Available small and large screens versions: .p-*-none, .pl-*-5, .p-*-10, .pt-*-5, .pt-*-10 (* - sm or md).

Margin

Margins are used to create space around elements, outside of any defined borders.

.m-none
.m-auto
.m-10
.mt-5
.ml-10

.m-none - none margin at all

.m-auto - margin auto - centered

.m-10 - 10px margin from all sides

.mt-5 - 5px margin from top

.ml-10 - 10px margin from left

Available small and large screens versions: .m-*-none, .m-*-auto, .m-*-10, .mt-*-5, .ml-*-10 (* - sm or md).
